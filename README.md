## Getting Started

### Executing program

To interact with the Degen Token smart contract using Remix, Metamask, and Snowtrace on the Avalanche FujiTest network, follow these step-by-step instructions:


1. Deploy the Contract:

- Open Remix IDE and deploy the Degen Token contract. Ensure Remix is connected to the Avalanche Fuji Testnet.
- Confirm the deployment transaction using Metamask.

2. Mint Tokens:

-As the contract owner, mint new tokens to designated addresses post-deployment.
-Use the mint function in Remix, specifying the recipient's address and the number of tokens to mint.

3. Transfer Tokens:

-Users can transfer tokens to different addresses using the transferTokens function.
-Enter the recipient's address and the number of tokens to transfer.

4. Redeem Tokens:

-Redeem tokens for in-game items via the redeemTokens function. Connect to the Avalanche Fuji Testnet in Remix for a realistic scenario.
-Provide the item ID for the in-game item you wish to redeem.

5. Check Token Balances:

-Users can view their token balances using the checkTokenBalance function in Remix.
-Input the address for which you want to verify the token balance.
